# ADR 001: Selección del Stack Tecnológico
[cite_start]**Fecha:** 18 de Abril de 2026 [cite: 32]
**Estado:** Aceptado

## Contexto
Necesitamos una tecnología que soporte el "Docs as Code" y despliegue rápido.

## Decisión
Se elige **Node.js** y **MkDocs** para la documentación automatizada.

## Consecuencias
[cite_start]Permite integrar GitHub Actions para que la documentación se convierta en un portal web con cada commit[cite: 50].
